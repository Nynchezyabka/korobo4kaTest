# Background Images

Папка для хранения фоновых изображений для таймера в разных категориях.

## С��руктура

Организуйте изображения по категориям:

- `security/` - Фоны для категории "Система безопасности"
- `mandatory/` - Фоны для категории "Обязательные дела"
- `joys/` - Фоны для категории "Простые радости"
- `accessibility-joys/` - Фоны для категории "Доступность простых радостей"
- `ego-joys/` - Фоны для категории "Эго-радости"

## Как добавлять новые фоны

1. Поместите изображение в соответствующую папку
2. Обновите массив в `script.js` с URL изображения из CDN Builder.io
3. Изображение должно быть в формате JPEG или PNG
4. Рекомендуемый размер: не менее 800px по ширине

## Текущие фоны

Все фоны в данный момент подгружаются с CDN Builder.io. При необходимости сохранения локальных копий просто добавьте файлы в соответствующие папки.
